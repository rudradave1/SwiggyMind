# SwiggyMind 🧠

> An AI ordering copilot that understands what you're craving — 
> not just what you type.

[GIF HERE — the single most important thing]

Built as a Swiggy Builders Club application. 
Demonstrates conversational food discovery on Android 
using KMP + Jetpack Compose + local/cloud LLM routing.

## What makes it different
Swiggy today requires you to know what you want.
SwiggyMind starts from intent — "something light under ₹200" —
and reasons its way to a recommendation with an explanation.
